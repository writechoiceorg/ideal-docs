Search Types

Individual

Searches for a specific individual.

Required Inputs

Individual

firstName, lastName, SSN, DOB

Additional Data Received

Status

The status of the individual search.

Full name

The individual's full name.

Last name

The individual's last name.

Second last name

The individual's second last name.

Name

The individual's name.

Birth date

The individual's birth date.

Birth municipality

The individual's birth municipality.

Gender

The individual's gender.

Father full name

The full name of the individual's father.

Mother full name

The full name of the individual's mother.

Death date

The individual's death date. Note:
  • Data on death events is limited to pre-2021 deaths in Puerto Rico.
  • We cannot validate death events that occurred after the stated limit (pre-2021) and outside of Puerto Rico.
